# 搜索框 Search · 组件设计规范

## 1. 概述

搜索框组件用于内容检索场景，提供从默认态 → 激活态 → 输入态的完整交互状态覆盖。共 6 种子组件，按状态和导航结构分类。

## 2. 子组件矩阵

| 编号 | 名称 | 状态 | 是否有返回按钮 | 输入框宽度 |
|------|------|------|--------------|-----------|
| A1 | 默认态 · 居中 | Default | 否 | 396px |
| A2 | 默认态 · 左对齐 | Default | 否 | 396px |
| B1 | 激活态 · 无返回 | Focused | 否 | 350px |
| B2 | 激活态 · 有返回 | Focused | 是 | 328px |
| C1 | 输入态 · 无返回 | Typing | 否 | 350px |
| C2 | 输入态 · 有返回 | Typing | 是 | 328px |

## 3. 视觉规范

### 3.1 容器
- **外容器**: 428 × 60px，背景 `white`，水平 padding 16px
- **输入框**: 高 36px，背景 `rgba(0, 81, 254, 0.04)`，圆角 12px

### 3.2 输入框宽度

| 场景 | 宽度 | 说明 |
|------|------|------|
| 默认态 (无取消) | 396px | 占满可用区域 (428 - 32) |
| 激活/输入态 (无返回) | 350px | 右侧留出"取消"按钮空间 |
| 激活/输入态 (有返回) | 328px | 左侧返回箭头 + 右侧取消按钮 |

### 3.3 图标

| 图标 | 尺寸 | 颜色 | 使用场景 |
|------|------|------|---------|
| 搜索图标 (默认/激活) | 18 × 18px | `#929296` | A1, A2, B1, B2 |
| 搜索图标 (输入态) | 18 × 18px | `#1A1A1A` | C1, C2 |
| 蓝色光标 | 1 × 18px | `#0099FF` | B1, B2, C1, C2 |
| 清除按钮 (圆形×) | 20 × 20px | `#929296` | C1, C2 |
| 返回箭头 | 24 × 24px | `#1A1A1A` | B2, C2 |

### 3.4 文字

| 元素 | 字号 | 字重 | 颜色 | 字体 |
|------|------|------|------|------|
| placeholder "搜索" | 17px | 400 | `rgba(60, 60, 67, 0.56)` | PingFang SC |
| 输入文本 "搜索" | 17px | 400 | `rgba(0, 0, 0, 0.90)` | PingFang SC |
| "取消" 按钮 | 16px | 400 | `#214CA5` | PingFang SC |

### 3.5 返回箭头布局
- 返回箭头容器: 12 × 24px，位于输入框左侧
- 箭头 SVG 24 × 24px，通过 `margin-left: -6px` 对齐
- 箭头与输入框间距: 12px

## 4. 状态说明

### A. 默认态 (Default)
- placeholder 文字 + 搜索图标
- A1: 图标和文字居中对齐
- A2: 图标和文字左对齐 (padding-left: 12px)

### B. 激活态 (Focused)
- 输入框获得焦点，显示蓝色光标
- placeholder 文字左对齐
- 右侧显示"取消"按钮
- B2: 额外显示左侧返回箭头

### C. 输入态 (Typing)
- 搜索图标变为 `#1A1A1A` (深色)
- 输入文字显示为 `rgba(0, 0, 0, 0.90)` (深色实文)
- 光标位于文字右侧
- 输入框右侧显示清除按钮 (圆形×)
- 右侧显示"取消"按钮
- C2: 额外显示左侧返回箭头

## 5. 交互行为

| 触发 | 行为 |
|------|------|
| 点击默认态输入框 | 进入激活态，显示光标和取消按钮 |
| 开始输入文字 | 进入输入态，搜索图标变深色，显示清除按钮 |
| 点击清除按钮 | 清空文字，回到激活态 |
| 点击取消按钮 | 回到默认态 |
| 点击返回箭头 | 返回上一级页面 |

## 6. Figma 属性映射

```
Search / state=default, align=center      → A1
Search / state=default, align=left         → A2
Search / state=focused, hasBack=false      → B1
Search / state=focused, hasBack=true       → B2
Search / state=typing, hasBack=false       → C1
Search / state=typing, hasBack=true        → C2
```
