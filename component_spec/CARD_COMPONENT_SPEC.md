# 卡片 Card 组件设计规范

## 1. 概述

卡片（Card）是承载结构化信息的容器组件，支持多种内容组合形式，适用于信息展示、图文推荐、数据呈现等场景。所有卡片共享相同的基础容器样式，通过内部区块组合形成不同变体。

## 2. 变体总览

| 编号 | 名称 | 总高度 | 核心结构 |
|------|------|--------|----------|
| A1 | 信息卡 | 564.5px | 头部行 + 标题 + 描述 + 金额 + 键值对 + 链接行×2 |
| A2 | 图文卡 | 252px | 封面图(180px) + 列表行(App图标+文字+按钮) |
| A3 | 沉浸图卡 | 296px | 全幅背景图 + 底部毛玻璃列表行(白色文字+白色按钮) |
| A4 | 图集卡 | 266px | 图集区(大图+四宫格) + 列表行(文字+按钮) |
| A5 | 多列表卡 | 197px | 头部行 + 列表行×2(App图标+文字+按钮) |
| A6 | 推荐卡 | 137px | 标题+描述+App图标 + 底部操作栏(描述+图标×3) |
| A7 | 数据卡 | 158.5px | 数据区(排名+步数) + 底部行(头像+标题+箭头) |

## 3. 基础容器

| 属性 | 值 |
|------|-----|
| 宽度 | 396px |
| 背景色 | `white`（Token `--分组背景色-Secondary`） |
| 圆角 | 24px |
| 溢出 | hidden |

## 4. 通用子区块

### 4.1 头部行（Header Row）
- **高度**：52px
- **内边距**：左右 16px
- **头像**：32×32px，圆形
- **应用名**：17px, Regular, `rgba(0,0,0,0.90)`
- **头像与文字间距**：8px

### 4.2 列表行（List Row）
- **高度**：72px
- **内边距**：左右 16px
- **App图标**：52×52px
- **图标与文字间距**：12px
- **列表文字**：17px, Regular, `rgba(0,0,0,0.90)`
- **描述文本**：12px, Regular, `rgba(60,60,67,0.76)`
- **按钮**：72×36px, 圆角9999px, 背景`rgba(116,116,128,0.08)`, 文字14px/Medium/`rgba(0,0,0,0.90)`

### 4.3 链接行（Link Row）
- **高度**：52px
- **内边距**：左右 16px
- **文字**：17px, Regular, `rgba(0,0,0,0.90)`
- **右侧箭头**：8×16px
- **顶部分割线**：宽364px, 0.5px, `rgba(0,0,0,0.08)`, 距左16px

### 4.4 分割线
- **宽度**：364px
- **高度**：0.5px
- **距左**：16px
- **颜色**：`rgba(0,0,0,0.10)`（Token `--描边分割色-border_standard`）

## 6. 设计 Token 映射

| Token 名 | 值 | 用途 |
|-----------|-----|------|
| `--分组背景色-Secondary` | `white` | 卡片背景 |

| `--描边分割色-border_standard` | `rgba(0,0,0,0.10)` | 分割线颜色 |
| `--分割色-Separators` | `rgba(0,0,0,0.08)` | 链接行顶部分割线 |

| `--透明填充色-Primary` | `rgba(120,120,120,0.20)` | A3毛玻璃背景 |
| `--图标色-icon_allwhite` | `white` | A3白色按钮背景 |

## 8. 使用场景

- **A1 信息卡**：账单详情、订阅扣费通知
- **A2 图文卡**：应用推荐、文章预览
- **A3 沉浸图卡**：游戏推荐、活动Banner
- **A4 图集卡**：相册分享、图片动态
- **A5 多列表卡**：应用更新通知、关联推荐
- **A6 推荐卡**：内容推荐、快捷操作入口
- **A7 数据卡**：运动排行、数据统计
