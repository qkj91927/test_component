# 卡片 Card · 组件设计规范

> **组件 ID**：`card`  
> **大类**：数据  
> **变体数量**：10 种子组件（C1 ~ C10）

---

## 1. 组件概述

卡片组件是一种承载结构化内容的圆角容器，用于在信息流、详情页等场景中展示多种组合形式的内容块。每张卡片由多个区块（头像行、图片区、标题区、辅助信息行、列表行、按钮区、Markdown 内容区等）自由组合而成，统一使用 24px 圆角白色容器，两侧缩进 16px（内容宽度 396px）。

## 3. 全局结构参数

### 3.1 卡片容器

| 属性 | 值 | 说明 |
|------|------|------|
| 外层宽度 | 428px | 设备宽度 |
| 内容宽度 | 396px | 左右各缩进 16px |
| 圆角 | 24px | 统一圆角 |
| 背景色 | `#FFFFFF` (bg_item) | 白色容器 |
| 溢出裁剪 | `overflow: hidden` | 图片撑满不超出 |

### 3.2 页面背景色约束

卡片组件为圆角白色容器，**页面背景色必须使用** `#F0F0F2` (`bg_bottom_standard`)，与卡片式列表(Grouped List)规则一致，白色卡片需要灰色底形成视觉层级分离。

## 5. 属性约束

1. **圆角统一**：所有卡片容器圆角必须为 24px，不可修改
2. **宽度固定**：卡片内容宽度 396px，两侧 16px 边距，不可调整
3. **图片比例**：短图 180px、长图 320px 为固定高度，宽度 396px 撑满
4. **头像行分割线**：仅底部展示分割线，无头像行时无此分割线
5. **九宫格布局**：1大+4小固定布局，大图必须在左侧，不支持其他排列
6. **辅助信息行**：始终在卡片底部，高度固定 40px
7. **大按钮**：大按钮类型仅为二级样式（灰色背景），不支持一级（蓝色）
8. **箭头行**：每行顶部必须有分割线，箭头为右侧 chevron_right 图标
9. **背景色**：页面使用卡片组件时，背景色必须为 `#F0F0F2`
10. **C10 高度自适应**：C10 卡片高度根据 Markdown 内容自适应，最大不超过 840px

## 6. C10 Markdown 内容规范

C10 卡片专用于承载 Markdown 渲染后的文本块内容，适用于 AI 生成内容展示、富文本详情页等场景。

### 6.1 可用文本块类型

C10 Markdown 内容区仅支持以下居左对齐（A 类）文本块子组件：

| 文本块编号 | 名称 | 字号 | 字重 | 行高 | 适用场景 |
|-----------|------|------|------|------|----------|
| A2 | 一级标题 | 22px | 500 | 28px | 卡片内章节标题 |
| A3 | 二级标题 | 18px | 500 | 26px | 卡片内子章节标题 |
| A4 | 正文段落 | 17px | 400 | 24px | 主要内容文本 |
| A5 | 前缀正文 | 17px | 400 | 24px | 有序/无序列表项 |
| A6 | 摘要文本 | 14px | 400 | 20px | 引用块、注释 |
| A7 | 提示文本 | 12px | 400 | 20px | 脚注、辅助说明 |

> **注意**：C10 卡片内**不使用 A1 大标题**，因为卡片头像行已提供标题，内容区从章节标题（A2）开始。

### 6.2 Markdown 语法映射

| Markdown 语法 | 映射的文本块 | 说明 |
|---------------|-------------|------|
| `## 二级标题` | A2 一级标题 | 卡片内顶级章节标题 |
| `### 三级标题` | A3 二级标题 | 卡片内子章节标题 |
| 普通段落 | A4 正文段落 | 主要内容 |
| `1. 有序列表` / `- 无序列表` | A5 前缀正文 | 列表项 |
| `> 引用` | A6 摘要文本 | 引用块 |
| 脚注 / `<small>` | A7 提示文本 | 辅助信息 |
| `[链接](url)` | Textlink | 蓝色链接文字 `#214CA5` |

### 6.3 文本块间距规范（卡片内）

卡片内文本块间距遵循紧凑原则，相比页面级文本块略有收紧：

#### 6.3.1 标题后间距

| 上方元素 | 下方元素 | 间距值 |
|----------|----------|--------|
| A2 一级标题 | A3 二级标题 | **12px** |
| A2 一级标题 | A4 正文段落 | **12px** |
| A3 二级标题 | A4 正文段落 | **8px** |
| A3 二级标题 | A5 前缀正文 | **8px** |

#### 6.3.2 正文/列表间距

| 上方元素 | 下方元素 | 间距值 |
|----------|----------|--------|
| A4 正文段落 | A4 正文段落 | **8px** |
| A4 正文段落 | A5 前缀正文 | **8px** |
| A5 前缀正文 | A5 前缀正文 | **4px** |
| A5 前缀正文 | A4 正文段落 | **8px** |

#### 6.3.3 辅助文本间距

| 上方元素 | 下方元素 | 间距值 |
|----------|----------|--------|
| A4 正文段落 | A6 摘要文本 | **12px** |
| A4 正文段落 | A7 提示文本 | **12px** |
| A6 摘要文本 | A7 提示文本 | **8px** |

#### 6.3.4 章节分隔间距

| 场景 | 间距值 |
|------|--------|
| 新章节（A2 一级标题前） | **20px** |
| 新子章节（A3 二级标题前） | **16px** |

### 6.4 Textlink 链接色

在 A4/A5/A6/A7 文本中可嵌入 Textlink 链接色文字：

| 属性 | 值 |
|------|------|
| 颜色 | `#214CA5` (`var(--text-link)`) |
| 字号 | 继承父容器 |
| 下划线 | 无 |
| 光标 | pointer |

### 6.5 推荐内容模板

**模板 A：AI 回答卡片**
```
[头像行：AI 助手标题]
──────────────────────
A2 一级标题（回答主题）
  └─ A4 正文段落（解释说明）
  └─ A4 正文段落

A3 二级标题（步骤/要点）
  └─ A5 前缀正文（1. 第一步）
  └─ A5 前缀正文（2. 第二步）
  └─ A5 前缀正文（3. 第三步）

A7 提示文本（了解更多：[帮助链接]）
```

**模板 B：内容详情卡片**
```
[头像行：作者/来源标题]
──────────────────────
A4 正文段落（正文内容）
A4 正文段落

A6 摘要文本（> 重点引用内容）

A4 正文段落（继续正文）
A7 提示文本（发布时间 · 阅读量）
```

**模板 C：说明/帮助卡片**
```
[头像行：功能名称]
──────────────────────
A4 正文段落（功能简介）

A3 二级标题（使用方法）
  └─ A5 前缀正文（• 方法一）
  └─ A5 前缀正文（• 方法二）

A3 二级标题（注意事项）
  └─ A4 正文段落（注意说明）

A7 提示文本（如有疑问，请[联系客服]）
```

### 6.6 禁止的组合

| 禁止模式 | 原因 |
|----------|------|
| A1 大标题 | 卡片头像行已提供标题，不可再用 A1 |
| A7 → A4 正文 | 提示文本应为末尾，后面不应有正文 |
| 连续多个 A2 | 同级标题不可连续，中间需有内容 |
| 居中对齐（B 类）| C10 卡片仅支持居左排版 |
