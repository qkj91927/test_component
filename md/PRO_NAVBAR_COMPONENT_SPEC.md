# 顶部导航栏Pro NavBarPro — 组件设计规范

> **组件层级**：高级组件 · 导航  
> **组件 ID**：`PRO_navbar`  
> **替代关系**：可替代默认组件"导航栏 NavBar"，提供更精简的导航样式

---

## 一、组件概述

顶部导航栏Pro 是高级组件库中的导航组件，采用**圆形图标按钮 + 透明背景**的设计语言，提供更现代化、沉浸感更强的页面导航体验。与默认 NavBar 的矩阵排列组合模式不同，NavBarPro 采用**枚举式**定义，共 13 种固定变体，分为 A.小导航（60px）和 B.大导航（72px）两大类。

### 设计特征

| 特征 | 说明 |
|------|------|
| 图标按钮 | 44×44px 圆形，背景 `rgba(120,120,128,0.08)` iOS 26 液态玻璃，圆角 9999px，`backdrop-filter: saturate(180%) blur(20px)` + `border: 0.5px solid rgba(255,255,255,0.6)` + `box-shadow: 0 0.5px 2px rgba(0,0,0,0.06), inset 0 0.5px 0 rgba(255,255,255,0.25)` |
| 图标尺寸 | 24×24px，居中于 44px 圆形按钮内（padding 10px） |
| 背景 | 透明，按钮自带淡灰色半透明毛玻璃效果，在白色/有色背景上均可见 |
| 资源路径 | `assets/CodeBuddyAssets/3566_3032/` |

| 元素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| A类标题 | 20px | 500 | `#333333`（Labels-Primary） |
| A类副标题 | 12px | 400 | `rgba(60, 60, 67, 0.76)`（text_secondary） |
| B类大标题 | 24px | 600 | `#333333`（Labels-Primary） |

### 3.5 文本渲染

标题和副标题直接使用 Color Token 作为文字颜色：

| 元素 | Color Token | 色值 |
|------|-------------|------|
| A类标题 / B类大标题 | `--Labels-Primary` | `#333333` |
| A类副标题 | `--text_secondary` | `rgba(60, 60, 67, 0.76)` |
| B类副标题 | `--text_secondary` | `rgba(60, 60, 67, 0.76)` |

```css
/* 标题 */
color: #333333;
/* 副标题 */
color: rgba(60, 60, 67, 0.76);
```

## 四、CSS 实现代码块

### 4.1 液态玻璃效果（iOS 26 Liquid Glass）

```css
/* 固定导航栏区域 — 吸顶容器（透明，毛玻璃在子元素上） */
.phone-navbar-fixed {
    flex-shrink: 0;
    width: 428px;
}

/* 图标按钮 — 44px 圆形液态玻璃 */
.pro-navbar-icon-btn {
    width: 44px;
    height: 44px;
    border-radius: 9999px;
    background: rgba(120, 120, 128, 0.08);
    backdrop-filter: saturate(180%) blur(20px);
    -webkit-backdrop-filter: saturate(180%) blur(20px);
    border: 0.5px solid rgba(255, 255, 255, 0.5);
    box-shadow: 0 0.5px 2px rgba(0, 0, 0, 0.06),
                inset 0 0.5px 0 rgba(255, 255, 255, 0.25);
}
```

> ⚠️ **iOS 26 液态玻璃设计**：使用 `rgba(120,120,128,0.08)` 淡灰色半透明背景 + 细边框 + 内外阴影，在白色背景上也能清晰可见玻璃质感。无需改变页面底色。

## 五、使用规则

1. **固定在页面顶部**：紧跟系统状态栏（StatusBar 54px）之下，不参与内容滚动
2. **液态玻璃效果**：图标按钮采用 iOS 26 液态玻璃设计（淡灰色半透明背景 + 细边框 + 内外阴影），在白色/有色背景上均可见
3. **图标必须替换**：SVG 资源为占位图标，实际使用时必须替换为真实图标
4. **不可与默认 NavBar 同时使用**：同一页面只能使用一种导航栏类型
5. **A类与B类不可同页混用**：同一页面只选择一种类型

## 五、与默认 NavBar 的差异

| 维度 | 默认 NavBar | 高级 NavBarPro |
|------|-------------|----------------|
| 变体数量 | 97 种（矩阵排列组合） | 13 种（枚举式） |
| 生成方式 | L×C×R 三维组合 + 约束过滤 | 固定变体列表 |
| 图标按钮样式 | 直接使用裸图标 | 44px 圆形容器 + iOS 26 液态玻璃 |
| 导航栏背景 | 白色 `#FFFFFF` + 底部分割线 | 透明，无分割线 |
| 大标题模式 | 不支持 | B类变体支持（24px/600） |
| 适用场景 | 通用页面导航 | 沉浸式、品牌化页面 |
